Contact

It'd be great to receive some feedback! Your observations on which features you find awkward or hard to use, or what is missing in the library, are essential to improve the library.

libtins is under heavy development, so any feature that you'd like included in the library could be implemented and shipped in the next release.

If you have any enqueries, or encounter any problems while using the library, feel free to ask on the following google group:

https://groups.google.com/forum/#!forum/libtins
