Detailed Description

template<typename OptionType, class Container = std::vector<uint8_t>>
class Tins::PDUOption< OptionType, Container >

Represents a PDU option field.

Several PDUs, such as TCP, IP, Dot11 or DHCP contain options. All of them behave exactly the same way. This class represents those options.

The OptionType template parameter indicates the type that will be used to store this option's identifier.

The Container template parameter indicates the container which will be used to store this option's data. The container must store data sequentially. std::vector<uint8_t> is the default container.
